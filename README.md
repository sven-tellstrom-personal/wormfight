# Snake

A single-player Snake game that runs in the browser.

## Play

Open `index.html` in any modern browser, or serve the folder locally:

```bash
npx --yes serve .
```

Then open the URL shown in the terminal (usually http://localhost:3000).

## Controls

- **Arrow keys** or **WASD** — move
- **Space** — start / pause / resume
- **Play** button — start or restart

Score increases by 10 per food eaten. Your best score is saved in the browser.
